LiftSync

A workout tracking backend built with Node.js, Express, and PostgreSQL.

Features

- Connects to a PostgreSQL database
- Retrieve all workouts through a REST API
- Add new workouts to the database
- Environment-based configuration using dotenv

Current API Routes

GET /
GET /test-db
GET /workouts
POST /workouts

Tech Stack

- Node.js
- Express
- PostgreSQL
